# Plegar Pro v18.2 Corrección Real Verificada

Esta versión corrige dos errores de ejecución de v18.1 que impedían mostrar las novedades:

1. El editor antiguo de pliegues intentaba utilizar un elemento eliminado.
2. El módulo Desarrollo aparecía en el menú, pero no tenía metadatos internos.

## Comprobación inmediata
Al abrir la aplicación debe aparecer una tarjeta verde con el texto:

**v18.2 · Cambios cargados y verificables**

Desde esa tarjeta se puede abrir directamente:
- Multiplegado dimensional.
- Crear desarrollo.
- Comparador V multiplegado.
- Simulación 2D.
- Simulación 3D.

Abre `ABRIR_PLEGAR_PRO.html` para evitar confundirla con copias anteriores.
